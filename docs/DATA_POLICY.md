# 数据与版权边界

## 开源仓库不应包含

- 完整投档线、逐分段表、位次表、招生计划、学费表。
- 从纸质书、PDF、Excel、网页批量整理出的完整结构化数据。
- 官方资料截图、扫描件、裁剪图、OCR 中间文件。
- 考生个人信息、真实方案、聊天记录或服务器凭据。

## 可以包含

- 数据结构说明。
- 少量示例数据。
- 数据清洗、构建、页面生成代码。
- 数据来源标记字段和免责声明。

## 页面必须展示的数据来源

建议至少展示：

- 投档分来源。
- 位次来源：官方最低位次、一分一段折算、人工估算等。
- 学费来源：招生计划专刊、学校章程、按层次估算等。
- 院校标签来源：官方名单、学科评估、人工初标等。

## 准确性声明

本项目用于志愿初筛和家庭讨论，不是官方填报系统，不承诺准确、完整、实时。正式填报应以省教育考试院、高校招生章程、招生计划专刊和官方填报系统为准。
